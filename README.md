# 송전선로 공사 현황 (songjeon-progress)

수원·용인 송전선로 지중화 건설공사의 **구간별·공정별 진행현황**을 매일 입력하고 한눈에 보는 대시보드 웹앱.

- 전체 7구간 / 총 연장 **3,448m** (수원 1~4구간 2,320m, 용인 5~7구간 1,128m)
- 개착식 7단계 사이클: 터파기 → ELP관 매설 → 차폐판 설치 → 되메우기 → 표층 메우기 → 가포장 → 본포장(완료)
- 6구간 특수: 개착 22m(7단계) + 강관압입 48m / 지향성압입 179m(별도)
- 체인 NO. 20m 간격 위치 관리

## 화면

| 화면 | 내용 |
|---|---|
| **대시보드** | 종합/본포장/수원/용인 KPI, 구간별 진행(공정 현황색·구간색 토글), 공정별 누적 합계, 현재 작업 현황 |
| **일일 입력** | 공정별 시작·종료 위치(체인 NO.+m) 입력 → 구간 자동 매핑, 계단식·범위 실시간 검증 |
| **노선도** | 지도 이미지 업로드 + 노선 보정(클릭) → 진행 현황 색 오버레이, 선단 마커 |
| **설정** | 구간 기준표, JSON/CSV 내보내기·가져오기, 입력 기록, 초기화 |

## 실행

빌드 불필요. 정적 파일이라 브라우저로 `index.html`을 열거나, 정적 서버로 서빙:

```bash
python -m http.server 4178
# http://localhost:4178
```

## 기술

- 순수 HTML + JavaScript + Tailwind CSS(CDN), 디자인 시스템 = Transmission Precision (`송전선로 스티치 디자인/DESIGN.md`)
- 데이터 저장: 브라우저 **localStorage** (백업은 설정 화면의 JSON/CSV 내보내기)
- 추후 다중 사용자 공유가 필요하면 Supabase 백엔드로 확장 (PRD 7장 참조)

자세한 요구사항은 [PRD](PRD_송전선로_공정관리.md) 참조.

## 알려진 단순화 (MVP)

- 6구간 개착(22m) 진행은 구간 범위 내 입력을 22m로 캡 처리(압입 위치와 정밀 분리는 미구현).
- 노선도 m↔픽셀 매핑은 폴리라인 픽셀거리 비례(정밀 측지 아님).
